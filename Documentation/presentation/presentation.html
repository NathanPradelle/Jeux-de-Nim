Présentation du projet



Règles du jeu



Dans ces jeux, on dispose sur la table entre les deux adversaires une collection d’objets identiques (jetons, cartes, allumettes…). Chaque joueur ou joueuse, à tour de rôle, retire un certain nombre d’objets (selon des règles à préciser). La première personne qui ne peut plus faire de coup est perdante (autrement dit, la personne qui retire le dernier objet a gagné) : c’est ce qu’on appelle le mode de jeu « normal ». On peut au contraire décider que le ou la joueuse qui retire le dernier objet a perdu : on appelle ce mode de victoire le mode « misère ». Dans ce qui suit, sauf mention du contraire, tous les jeux sont décrits en mode de victoire normal.

Comment jouer

Pour jouer, rien de plus simple !

Une fois le programme lancé, vous serez accueilli par l'écran titre. Vous aurez le choix entre directement jouer avec les paramètres par défaut ou modifier les paramètres.



Pour sélectionner des allumettes, vous avez simplement à cliquer dessus ! Vous pouvez désélectionner une allumettes avec un clique droit.
Pour valider votre sélection et retirer les allumettes, vous devez appuyer sur la touche "Espace".






Les paramètres



Le bouton "Thème" permet de naviguer entre plusieurs thèmes facilement modifiables !
Le bouton langue permet lui de passer de français à anglais à japonais, il est aussi très simplement modifiables.

Le mode misère est activable ou désactivable facilement à l'aide du bouton "Mode misère".

Le bouton "Mode de jeu" permet quant à lui de passer d'un jeu à joueur contre joueur à un jeu contre l'ordinateur. Ici, l'ordinateur jouera ses coups aléatoirements.

L'entrée "Objets" permet de mettre le jeu en mode 1 lignes avec un nombre choisi d'allumettes. Pour ce faire, vous avez simplement à entrer le nombre d'allumettes que vous désirez avoir.

L'entrée "Tirages" elle, permet de choisir le nombre maximum d'allumettes pouvant être retirées en un coup.

Le bouton "Marienbad" permet de modifier les paramètres du mode de jeu "Marienbad", en plusieurs lignes donc.


Pour sauvegarder les paramètres, vous pouvez simplement retourner sur l'écran titre.






Paramètres Marienbad





Le nombre maximal de lignes est limité à 20. Vous pouvez choisir le nombre d'allumettes que vous désirez dans une ligne en modifiant l'entrée associée à cette ligne.
Une fois fait, vous pouvez sauvegarder et vous assurer qu'aucune erreur n'est présente.






Selection des noms



Une fois vos paramètres entrés, vous êtes paré à jouer ! Sur l'écran titre, appuyez sur le bouton "Jouer" pour afficher le menu de sélection de nom.



Sur cet écran, vous pouvez sélection un nom, en l'entrant dans la barre d'entrée, et une couleur, en cliquant sur votre nom !
Attention, une fois votre nom défini, il n'est plus possible de changer, donc choisissez bien!
